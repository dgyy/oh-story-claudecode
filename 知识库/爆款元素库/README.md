## 设定元素

### 金手指类型
- **系统面板**：属性、技能、商城、任务（七猫系统1193标签）
- **随身空间**：独立存储/修炼空间
- **预知未来**：重生记忆、预言能力（七猫重生869标签）
- **复制/模拟**：模拟对战、复制技能
- **数据化**：将现实数据化便于操作
- **时间系**：暂停、倒流、加速
- **召唤系**：召唤历史人物/虚构角色
- **鉴定系**：看到隐藏信息

### 世界观模板
- **学院流**：学校背景的升级故事
- **宗门流**：修仙宗门中的成长
- **都市异能**：隐藏在现代的超能力（番茄都市高武10本）
- **末世生存**：丧尸/异变/灾难后（黑岩末世5本）
- **星际争霸**：太空时代的战争
- **历史架空**：架空历史朝代（番茄历史脑洞7本）
- **虚拟现实**：游戏世界中的冒险（番茄动漫衍生20本）

## 命名规律（数据验证）

### 短篇命名公式（跨平台热门作品分析）

**公式：`[极端事件] + [被逼/死亡] + [反击手段]`**

高频词统计：
- 疯了/杀疯了/悔疯了 → 17次（极端情绪词）
- 我反手/我让他 → 4次（反击型）
- 全家/所有人都 → 4次（集体压迫）
- 清明节/愚人节/除夕夜 → 4次（时间节点开场）

示例：
- `白眼狼妹妹把家产送人，我让她净身出户`
- `亲妈逼我上台后我死在聚光灯下`
- `剖腹产第七天我死在厨房`

### 长篇命名公式（番茄 7248 本分析）

高频词：快穿、系统、一见钟情、离婚后、反派、开局秒杀

示例：
- `病娇妈妈们找上门，儿女都吓哭了`
- `因为长得太美，被系统当女神绑定`
- `黑化后，我终成毁灭世界的魔神`

### 知乎命名风格（8142 篇分析）

平均7.9字，比黑岩/点众短得多，更含蓄：
- `除夕夜，婆婆让我老公兼祧两房`（事件型）
- `夫君他柔弱不能自理`（反差型）
- `弹幕叫我远离男主，我偏不`（互动型）
- `花间辞`（文学型，仅3字）
