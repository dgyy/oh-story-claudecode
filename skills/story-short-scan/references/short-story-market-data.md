## 开头模式速查（跨平台验证）

### 模式1：医疗被拒（高频）
```
我刚做完XX手术，医生再三嘱咐要静养。
妈妈/婆婆/老公却强行让我做XX。
"就你娇气，别人怎么不这样？"
```
平台分布：黑岩/点众高频、知乎偶发

### 模式2：财产被夺（高频）
```
XX头七刚过，XX就在家族群里@我。
"你那XX，是不是该交出来了？"
"你一个女孩子家，要那么大XX有什么用？"
```
平台分布：黑岩/点众高频

### 模式3：情感背叛（中频）
```
第一次发现丈夫和XX在XX，是在我们的婚房里。
/ 我被关进精神病院的第X年，XX终于来找我。
```
平台分布：全平台通用

### 模式4：亲情牺牲被辜负（中频）
```
爹娘凑到的钱只够赎一人，他们毫不犹豫的救了XX。
"你XX，怎么能进那种地方？"
```
平台分布：黑岩/七猫高频、知乎低频

### 模式5：对话炸弹（番茄长篇专用）
```
"儿子，别舔了，爹带你去见见世面！"
/ 【叮！检测到宿主颠覆式育儿，年少轻狂+1】
```
平台分布：番茄/七猫长篇高频
