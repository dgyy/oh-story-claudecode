## 开篇核心法则

### 黄金一章

- 从故事最精彩、最有冲突的地方写起
- 主角三种状态选一：身处危机 / 令人羡慕 / 被丢入陌生环境
- 300字内主角登场，1000字内出现爽点或期待点

### 冲突节奏

- 故事本质是冲突，冲突不能断也不能过密
- 两个紧张场景间要有缓和，但缓和不等于无冲突
- 致命错误：开头大段背景介绍全是"缓"没有"冲突"
- 正确做法：背景融入冲突（《斗破苍穹》用旁人议论萧炎同时完成背景和冲突）

### 开篇结构范例（鬼灭之刃）

| 阶段 | 内容 | 功能 |
|------|------|------|
| 起 | 冒雪卖炭+家庭日常+嗅觉帮人+对话引世界观 | 四板斧立人设+嵌入主线 |
| 转 | 惨案→妹妹鬼化→义勇出现→跪地求饶→智慧反击 | 危机叠加+获得动力和目标 |
| 合 | 留下短/中期目标+凄冷景色渲染+牵亲人奔向未知 | 新篇章开启 |

### 信息团排版

- 不要第一章塞入所有设定，分批释放
- 优先级：危机感 > 人设 > 金手指暗示 > 世界观
- 每次释放服务于当前情绪目标

### 情绪波动线（实操）

上行-下行交替法：穿越成仙人（上行）→ 只是记名弟子（下行）→ 杂活累活还背锅（下行到最低+共鸣点）→ 有金手指了（上行）→ 金手指是同归于尽的（下行+戏剧性）→ 和穿越前有什么区别（触底）

关键：每次波动要有具体信息支撑；共鸣是关键步骤；戏剧性情节必须写出来。

### 信息差与期待感

- 期待感公式：情绪 + 情节（人设+动机）+ 信息差，三者缺一则不足
- 核心循环：情报→收获→震惊，短平快过渡到下一个有用剧情

### 书名/简介/开篇三位一体

- 书名告诉读者金手指和题材
- 简介只写三件事：主角处境、金手指能干嘛、第一个爽点预览
- 开篇必须呼应书名和简介的承诺
- 书名翻车预警：三字四字看不出亮点是大忌；不要拿悬念当期待感，直接告诉能干嘛

### 循环套路

| 类型 | 循环模式 |
|------|----------|
| 升级文 | 展示优势→制造信息差→展示信息差→装逼震惊→收获奖励→铺垫下文 |
| 日常/狗粮文 | 稳定人设+固定剧情流程=读者稳定期待（例：爱吃醋女主+偷家=反复利用） |
| 高武/升级文 | 练武→拜师→比赛，靠升级循环的稳定期待感拉住读者 |

## 开头常见问题速查

### 致命问题

| 问题 | 症状 | 修正 |
|------|------|------|
| 信息传达不清 | 主角在哪/干什么/动机不明 | 开局交代背景和处境，明确目标再进事件 |
| 人设逼格过低 | 见美女紧张/无意义内心戏/爆粗口 | 主角要有b格，配角可以奇形怪状但要有特色 |
| 情绪缺失 | 战斗枯燥/看几段不想读 | 战斗中穿插情绪刻画——恐惧、愤怒、决心 |
| 节奏问题 | 出场过多无关人物/铺垫太长 | 删无关互动，尽快进主线 |
| 金手指出场突兀 | 像写日记一样冒出金手指 | 用危机或困境触发金手指 |
| 铺垫章掉追读 | 章节名暴露是铺垫 | 用有戏剧性的标题把读者"骗"进来 |

### 改进方向

- **强情绪开头**：从高潮事件或复仇类强情绪切入，重大事件不能写得像闹着玩
- **情报与期待感**：每个情报都要有期待价值，关键情报拉起"然后呢？"
- **目标驱动**：主角目标要交代清楚，打斗必须基于主角目标
- **共鸣**：通过角色处境产生共鸣，先让读者站在主角这边

### 金手指与人设匹配

| 人设类型 | 适合的金手指 | 翻车组合 |
|----------|-------------|----------|
| 开朗/搞笑型 | 自爆类（装逼爽+吐槽契合） | 搞笑型+自爆金手指=不搭 |
| 心狠手辣型 | 传统型（升级、资源获取） | 冷酷型+搞笑金手指=违和 |

### 创意的正确展开

- 有创意不等于写好了——必须把创意点变成剧情，用对比放大感觉
- 书名/简介/正文三者统一（书名写"苟"但主角张扬=跑题）

### 内容红线

- 擦边和三观不正风险极大，非要写用外国背景+提前跟编辑打招呼
- 女频特殊规则：双洁是基础规则，不要挑战；补救用"其实是同一个人（转世/穿越）"化解

## 黄金开篇技巧

- **开头视角**：主角视角切入，建立"我在经历这件事"的沉浸感
- **开头要有具体事件**：第一件事就要有事情发生，不是世界观说明书
- **三种开局方法**：危机开局（生死压力）/ 悬疑开局（悬念钩好奇心）/ 反转开局（颠覆预期）
- **金手指时机因题材而异**：系统流第一章必须出，其他题材可延后（关键是金手指前剧情本身够吸引）
- **开头节奏因题材而异**：不同题材读者耐心不同，不能一刀切

## 三大基点与核心模板

### 三大基点

| 基点 | 内容 | 作用 |
|------|------|------|
| 人设基点 | 展示主角核心性格和处境 | 建立代入感和共情 |
| 切入点基点 | 主角遭遇的第一个冲突/机遇 | 制造紧迫感和阅读动力 |
| 金手指基点 | 展示主角独特优势 | 制造期待感和差异化 |

三个基点前3章内必须全部完成。困境型先人设，优势型先金手指。切入点必须跟主线强相关。

### 模板一：困境切入型

1. 主角处于不利处境（人设基点=共情入口）
2. 遭遇压迫/危机（切入点=紧迫感）
3. 金手指激活/展示（金手指=反转期待）
4. 初步解决问题但引出更大问题（钩子）

### 模板二：优势切入型

1. 主角已有独特优势但被低估/隐藏（人设+金手指合一）
2. 优势在某个场景中意外暴露（切入点）
3. 周围人的认知被刷新（震惊反应）
4. 暴露引来更大的关注/麻烦（钩子）

## 题材开头数据库

> 按题材分类的开头写法参考：真实范例+结构分析+可套用模板。

### 都市脑洞

范例：《病娇妈妈们找上门》 | 结构：场景→独处→意外→反转→系统

```
{城市名}。
{简短场景描述}。
{主角}{一个反常行为}。
{一句话解释为什么这么做，但留悬念}。
实际上，{真相，制造更大悬念}。
```

### 豪门总裁

范例A：《禁她入骨》 | 结构：极端环境→感官描写→身份交代→被卖真相

```
{极端/异国地点}。{季节}。
{2-3个感官细节}。
{困境描写，1-2句}。
{主角身份交代}。
{一句话揭示谁害了她}。
```

范例B：《不愿再让你低头》 | 结构：医院→诊断结果→倒计时→电话

```
出了{医院/法院/某场所}，{天气映射心情}。
{主角}{一个微动作暗示内心}。
脑海中闪过{某人}的话——
"{冲击性的一句话}"
{主角的反应}。
{一个动作推动下一步}。
```

### 古代宅斗

范例：《十年前他们要我顶替妹妹进青楼》 | 结构：核心事件→被弃→回归→反差

```
{N年前，被害/被弃的事件}。
{亲人选择救别人不救主角}。
"{被弃时的伤人台词}"
{N年后，主角已{隐藏身份/实力}}。
{回归场景，制造身份反差}。
```

### 玄幻修仙

范例A：《西游你叫我吗喽》 | 结构：世界危机→日常切入→反差主角→系统/穿越

```
"{一条震惊的新闻/传闻}"
"{群众反应/讨论}"
——安静——
{场景转换：学校/公司/日常}
{权威人物宣布一个危险任务}
{所有人退缩}
"我报名！"
{主角站出来，所有人震惊}
{暗示主角有隐藏优势}
```

范例B（动漫衍生）：原作角色互动→穿越者内心→系统选择

```
{原作角色名}的声音{描述}
{原作角色向主角提出请求/挑战}
主角内心：{穿越者视角的分析}
系统/面板出现：
【选择一：{选项}】奖励：{xxx} 代价：{xxx}
【选择二：{选项}】奖励：{xxx} 代价：{xxx}
主角选择+理由。
```

### 青春校园

范例：《你携盛夏，渡我寒冬》 | 结构：苦难叠加→遇见→改变

```
最{负面状态}那年，我遇到了{对方}。
我{被欺负}，被{家人伤害}，被{社会歧视}。
{对方}则{不良/叛逆行为列举}。
我走投无路。
"{交易/请求台词}"
"{对方的反应，出人意料}"
```

### 现实情感

范例：《五年恩情，一朝两清》 | 结构：纪念日被放鸽子→揭穿→回忆契约

```
{对方}{重要事件}那天，是{我们的重要日子}。
{被忽视的场景细节，暗示等待}。
手机屏幕亮了："{敷衍的消息}"
{主角查看{朋友圈/聊天}——真相}。
{一句话内心独白，冷静但心碎}。
```

### 点众轻喜剧

范例：《捡回来的小乞丐，竟然想咬我》 | 结构：吐槽→捡人→反差→笑料→升级

```
{主角}看着{危险/奇怪的事物}，{非正常反应}。
"{吐槽台词}"
{世界观一句话交代}
{对方的真实身份，与表面反差极大}
{对方的攻击完全无效→制造笑点}
{意外发现对方获得好处→双赢设定}
```

### 四合院/年代文

范例：《四合院：杀伐果断，开局众禽上刑场》 | 结构：穿越→回忆原主悲惨→复仇计划

```
{主角}这会儿{状态}，{反应}。
明明{昨天在做的事}，一{闭眼/醒来}，居然就{穿越了}。
{记忆涌入——原主的悲惨经历}
"{情绪爆发——愤怒/不甘}"
利用{前世知识/穿越者优势}，{第一个行动目标}
```

## 开头通用规则

### 基本要求

- 主要角色（主角+可攻略+主要配角+主要反派）第一个一级结构内不得登场3个以上
- 开头以叙事为主，压缩介绍性/抒情性/环境描写
- 人设通过叙事手法体现，叙事优先于塑造人设
- 正叙为主，不插叙/不切视角/不用回忆梦境/不大篇幅对话
- 不写序章/楔子/引（除非叙事为主、足够简短且与第一章逻辑紧密）
- 第一章由主角行动引导走向，主角不能"打酱油"
- 世界观详细解说至少等到第一个一级结构结束

### 布局要求

- 切入点（转折点）在第一个一级结构中出现，最好第一章
- 情绪：尽快迎来第一次上行（推荐"先短暂下行再上行"或"原地起飞"）
- 第一个冲突影响重大，事先点明解决的好处和不解决的坏处
- 主角行动必须带来正面收益
- 第一章必须说明：主角目标+本文卖点

### 期待感三路径

- 事业线：设定新颖，读者预见结局但猜不到过程
- 爱情线：人设搭配有吸引力，开头抛出亮点
- 双线缠绕：事业线与爱情线同时进行，互相影响

### 噱头三分法

| 噱头类型 | 写法 | 要点 |
|----------|------|------|
| 事件噱头 | 书名即事件，极快进入故事 | 用日常常见事件开局；一上来就进事件，不要铺垫穿越/金手指 |
| 金手指噱头 | 主角→困境→目标→金手指首次施展→少许成果 | 可结合事件写法：事件拉成主线→诞生目标→出金手指 |
| 人设噱头（高阶） | 出彩人设直接影响剧情构建 | 杀伐果断不算出彩人设；新人不推荐 |

噱头吸引读者后立刻嫁接主线，事件结束后马上接新目标。

### 七步法字数分配参考

| 段落 | 字数 | 内容 |
|------|------|------|
| 世界观+穿越事实+前身经历 | 300字 | 快速交代背景 |
| 第一个吸睛小剧情 | 500字 | 用事件制造初始吸引 |
| 前置金手指获得安全感 | 400字 | 金手指首次展示 |
| 感受到不安全 | 400字 | 新的威胁出现 |
| 遇见危机 | 500字 | 核心冲突到来 |
| 直接使用金手指 | 400字 | 解决问题，完成第一个爽点 |

## 卖点设计与验证

### 卖点的显性与隐性分层

- 商业性卖点（噱头、创新、新鲜金手指）vs 文学性卖点（文风、笔力、世界观）
- 卖点被"海水"淹没=需要更多努力（番茄波涛最大，起点次之）
- 即使大神也需要多卖点叠加

### 五步验证法

1. 检查核心梗/金手指/开头情节（不能以"升级""爽文"为卖点）
2. 围绕核心特化世界观和情节
3. 检查人设反差/卖点
4. 检查个人写法特色
5. 蹭热点/蹭话题

## 平台差异化

| 维度 | 番茄 | 起点 | 晋江 |
|------|------|------|------|
| 核心取舍 | 节奏 > 代入感 | 代入感 > 节奏 | 人设 > 情节 |
| 开头策略 | 强情绪+强噱头 | 人物铺垫+慢热可接受 | 角色魅力先立 |
| 期待感 | 核心情绪深挖+反复提及 | 多线铺垫+伏笔 | 情感推进即期待 |
| 爽点密度 | 每2000字一个 | 每3000-5000字一个 | 情感推进即爽点 |
| 节奏 | 快，不能停 | 可慢，不能散 | 细，不能水 |
