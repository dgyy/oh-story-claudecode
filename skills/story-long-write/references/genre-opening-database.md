# 题材开头数据库

> 从大量热门网文的第一章提炼，按题材分类的开头写法参考。
> 每个题材提供：真实开头范例 + 结构分析 + 可直接套用的模板

---

## 一、都市脑洞

### 范例：《病娇妈妈们找上门，儿女都吓哭了》
**开头结构**：场景→独处→意外→反转→系统

> 江城市。破旧楼房内。苏牧下班后没回家。他跟老婆江亦瑶撒了个谎，说要出差。实际上，他躲到了这里。

**结构拆解**：
1. 2字交代地点（江城市）
2. 5字交代场景（破旧楼房内）
3. 一句话交代状态（下班没回家）
4. 一句话制造悬念（撒谎，躲）
5. 然后展开：意外女人出现→仙人跳→离婚→系统激活

**可套用模板**：
```
{城市名}。
{简短场景描述}。
{主角}{一个反常行为}。
{一句话解释为什么这么做，但留悬念}。
实际上，{真相，制造更大悬念}。
```

## 二、豪门总裁

### 范例A：《禁她入骨》
**开头结构**：极端环境→感官描写→身份交代→被卖真相

> 东南亚，帕孔地区。盛夏。烈日高悬，炙烤大地。空气又干又烫，每一口呼吸都如在吞咽含沙的热风。四周散发着尿液的骚臭味，汗液的酸腐味。

**结构拆解**：
1. 地点+季节（东南亚，盛夏）
2. 温度+嗅觉感官
3. 困境（铁笼，被当猪仔）
4. 主角身份（大二学生，被骗）
5. 背叛者揭晓（亲叔叔）

**可套用模板**：
```
{极端/异国地点}。{季节}。
{2-3个感官细节}。
{困境描写，1-2句}。
{主角身份交代}。
{一句话揭示谁害了她}。
```

### 范例B：《不愿再让你低头》
**开头结构**：医院→诊断结果→倒计时→电话

> 出了医院，浓重的夜色压下来，秋风卷着落叶吹来，刚从医院出来的岳绮颜下意识的拢了拢风衣，握着诊断书的手指因为用力骨节泛出青白色。

**结构拆解**：
1. 医院出口（暗示坏消息）
2. 天气描写映射心情
3. 握诊断书的动作（不说内容，制造悬念）
4. 医生话引用（最多两年半）
5. 打电话给丈夫（核心关系）

**可套用模板**：
```
出了{医院/法院/某场所}，{天气映射心情}。
{主角}{一个微动作暗示内心}。
脑海中闪过{某人}的话——
"{冲击性的一句话}"
{主角的反应}。
{一个动作推动下一步}。
```

## 三、古代宅斗

### 范例：《十年前他们要我顶替妹妹进青楼》
**开头结构**：核心事件→被弃→回归→反差

> 债主把我和妹妹卖去西域青楼抵债。爹娘凑到的钱只够赎一人，他们毫不犹豫的救了妹妹。

**结构拆解**：
1. 一句话核心设定（被卖）
2. 父母选择（救妹妹不救我）
3. 十年后对比（妹妹入东宫 vs 我成摄政王）
4. 回归场景（走正门）
5. 父母反应（嫌弃→恐惧）

**可套用模板**：
```
{N年前，被害/被弃的事件}。
{亲人选择救别人不救主角}。
"{被弃时的伤人台词}"
{N年后，主角已{隐藏身份/实力}}。
{回归场景，制造身份反差}。
```

## 四、玄幻修仙

### 范例A：《西游你叫我吗喽》
**开头结构**：世界危机→日常切入→反差主角→系统/穿越

> "听说了吗，樱花国京都，被一只青毛巨狮占据……"

**结构拆解**：
1. 群众对话交代世界观
2. 危机背景（西游妖魔入侵现实）
3. 学校场景（日常反差）
4. 主角报名（所有人觉得他疯了）
5. 穿越者优势（知道西游记剧情）

**可套用模板**：
```
"{一条震惊的新闻/传闻}"
"{群众反应/讨论}"
——安静——
{场景转换：学校/公司/日常}
{权威人物宣布一个危险任务}
{所有人退缩}
"我报名！"
{主角站出来，所有人震惊}
{暗示主角有隐藏优势}
```

### 范例B：《长生火影》（动漫衍生）
**开头结构**：原作角色互动→穿越者内心→系统选择

**可套用模板**：
```
"{原作角色名}的声音{描述}"
{原作角色向主角提出请求/挑战}
主角内心：{穿越者视角的分析}
系统/面板出现：
【选择一：{选项}】奖励：{xxx} 代价：{xxx}
【选择二：{选项}】奖励：{xxx} 代价：{xxx}
主角选择+理由。
```

## 五、青春校园

### 范例：《你携盛夏，渡我寒冬》
**开头结构**：苦难叠加→遇见→改变

> 最走投无路那年，我遇到了林燃。我被同学欺负，被亲妈揍，被邻居们骂野种。

**结构拆解**：
1. 倒叙钩子（最走投无路那年）
2. 三重苦难叠加（欺负+家暴+歧视）
3. 遇见（他/她出现）
4. 第一印象（不良少年形象）
5. 交易（我帮你写作业，你保护我）

**可套用模板**：
```
最{负面状态}那年，我遇到了{对方}。
我{被欺负}，被{家人伤害}，被{社会歧视}。
{对方}则{不良/叛逆行为列举}。
我走投无路。
"{交易/请求台词}"
"{对方的反应，出人意料}"
```

## 六、现实情感

### 范例：《五年恩情，一朝两清》
**开头结构**：纪念日被放鸽子→揭穿→回忆契约

> 顾泽言的白月光林薇回国那天，是我们结婚三周年纪念日。我坐在餐厅靠窗的位置，看着桌上冷透的牛排和燃尽的蜡烛。

**结构拆解**：
1. 两个事件叠加（纪念日 vs 白月光回国）
2. 场景细节（冷牛排、燃尽蜡烛=被遗忘）
3. 手机消息（"临时有急事"）
4. 朋友圈验证（白月光发图）
5. 契约婚姻背景交代

**可套用模板**：
```
{对方}{重要事件}那天，是{我们的重要日子}。
{被忽视的场景细节，暗示等待}。
手机屏幕亮了："{敷衍的消息}"
{主角查看{朋友圈/聊天}——真相}。
{一句话内心独白，冷静但心碎}。
```

## 七、点众轻喜剧

### 范例：《捡回来的小乞丐，竟然想咬我》
**开头结构**：吐槽→捡人→反差→笑料→升级

> 许寂看着雪地里那个浑身是血、眼神凶得像要吃人的小罗莉，叹了口气。"这年头，碰瓷的成本都这么高了吗？"

**结构拆解**：
1. 主角吐槽式反应（不是害怕是吐槽）
2. 建立世界观（修仙+穿越10年+没金手指）
3. 捡到危险生物（小萝莉=鬼帝重生）
4. 认知错位制造笑点（鬼帝咬不动凡人）
5. 一碗粥引发升级（种田天赋+灵气）

**可套用模板**：
```
{主角}看着{危险/奇怪的事物}，{非正常反应}。
"{吐槽台词}"
{世界观一句话交代：穿越N年/系统/金手指状态}
{捡到/遇到一个人/物}
{对方的真实身份，与表面反差极大}
{对方的攻击完全无效→制造笑点}
{主角毫无危机感地照顾对方}
{意外发现对方获得好处→双赢设定}
```

## 八、四合院/年代文

### 范例：《四合院：杀伐果断，开局众禽上刑场》
**开头结构**：穿越→回忆原主悲惨→复仇计划

> 李建业这会儿脑袋嗡嗡的，整个人都蒙了。明明昨晚只是熬夜追了个剧，一闭眼再睁开，居然就换了个地儿。

**可套用模板**：
```
{主角}这会儿{状态}，{反应}。
明明{昨天在做的事}，一{闭眼/醒来}，居然就{穿越了}。
{记忆涌入——原主的悲惨经历}
{父母被害/被欺负/被冤枉的经过}
"{情绪爆发——愤怒/不甘}"
{立刻制定复仇计划}
利用{前世知识/穿越者优势}，{第一个行动目标}
```

## 开头选择决策树

```
你的故事是？
├── 女频言情
│   ├── 总裁豪门 → 用"被迫+极端环境"或"纪念日+白月光"
│   ├── 古代宅斗 → 用"被害+回归+身份反差"
│   └── 青春校园 → 用"苦难叠加+遇见"
├── 男频爽文
│   ├── 都市脑洞 → 用"中年危机+系统"
│   ├── 玄幻修仙 → 用"群众恐慌+主角报名"
│   └── 动漫衍生 → 用"原作场景+系统选择"
├── 现实题材
│   ├── 家庭伦理 → 用"场景冲突+被背叛"
│   └── 行业文 → 用"专业场景+异常事件"
└── 轻喜剧
    └── → 用"吐槽式反应+认知错位"
```
